4.3.2.10 SOLUTION CAPABILITY MATRIX
A solution capability matrix is a model that provides a simple visual way to examine capabilities and solution
components in one view, identifying where capabilities will be addressed in the new solution. When timing information
is added, the product team can use the model during planning discussions to understand when different solution
components are expected to be delivered.
