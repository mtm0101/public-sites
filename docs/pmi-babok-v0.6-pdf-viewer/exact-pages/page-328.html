9.1.2.4 PRIORITIZATION SCHEMES
Prioritization schemes are different methods used to prioritize requirements, features, or other product information.
For the purposes of evaluating solution performance, prioritization schemes can be used to rank the value of the
beneﬁts obtained from the solution as well as the severity of any challenges it creates. Doing so can provide a way
to consider whether the beneﬁts outweigh the challenges and vice versa. For more information on prioritization
schemes, see Section 7.7.2.5.
