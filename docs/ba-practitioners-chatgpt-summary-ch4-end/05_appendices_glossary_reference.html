# Appendices and Glossary Reference (Phụ lục và thuật ngữ tham chiếu)

Scope note: The remaining back matter is handled as a study reference rather than a reproduced copy. This page explains how to use the contributors/reviewers appendix, bibliography, and glossary responsibly when studying Chapters 4-6.

Learning map (Bản đồ học tập)

  • Back matter can improve study when used as a reference map rather than copied content.
  • Glossary terms should be learned through examples, artifacts, and decisions.
  • Bilingual terminology should preserve distinctions among related BA concepts.
  • The bibliography can guide deeper research into business analysis and project management practice.
  • The source index is excluded from the generated study companion.

# X1 Appendix X1 - Contributors and Reviewers (Phụ lục X1 - Người đóng góp và phản biện)

Source page anchor: p. 177 · direct anchor

In practice, Appendix X1 - Contributors and Reviewers should be used as a reference aid rather than a copied body of learning content. The study value is in connecting the reference material to practical business analysis decisions.

Use it when

Use this reference page when checking terminology, planning further reading, or building a study map for the remaining chapters.

How to apply

Pick a term or source, connect it to an artifact from Chapters 4-6, and write a short example showing how it changes a project decision.

Practical example

A learner studies "baseline" by linking it to a requirements set, approval record, change request, and traceability matrix entry. The term becomes a working concept rather than a memorized definition.

Common mistakes / anti-patterns

  • Memorizing terms without examples
  • Using literal translation that changes meaning
  • Copying reference material instead of learning how to apply it

Checklist

  • Map each term to a project artifact
  • Keep English term with Vietnamese explanation
  • Use examples to test understanding
  • Consult the original source for full official context

Key takeaway: Use Appendix X1 - Contributors and Reviewers when it helps the team make a clearer decision, reduce uncertainty, or protect business value.
