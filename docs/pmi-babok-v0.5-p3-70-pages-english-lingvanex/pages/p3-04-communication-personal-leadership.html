X3.3 Communication Skills (Kỹ năng giao tiếp)

What it is

The ability to create shared understanding through listening, speaking, writing, visuals, facilitation, and relationship management. For section X3.3, treat the material as a practical aid that complements professional judgment rather than replacing it.

Why it matters

For section X3.3, the main value is turning professional knowledge into usable BA work products while improving review discipline and decision confidence.

When to use it

Use communication skills across every BA activity because analysis results only create value when others understand, trust, and act on them. Use section X3.3 when the situation contains ambiguity, cross-functional dependency, quality risk, or a need to explain recommendation logic.
